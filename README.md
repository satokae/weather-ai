# Weather AI

## データ

データ出典：気象庁ホームページ [過去の気象データ](https://www.data.jma.go.jp/risk/obsdl/#)

### 地点

- 浜松

### 項目

- 気温
- 露点温度
- 風向・風速
- 現地気圧
- 天気

### 期間（日本標準時）

- 2023年1月1日から
- 2023年12月31日まで

- 2024年1月1日から
- 2024年12月31日まで

- 2025年1月1日から
- 2025年12月31日まで

の時別値を表示

### オプション

- 利用上注意が必要なデータを表示させない
- 観測環境などの変化以前のデータを表示させない
- ダウンロードデータはすべて数値で格納
